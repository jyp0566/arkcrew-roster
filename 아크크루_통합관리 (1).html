🔥 Firebase 연결 설정 (최초 1회)

아래 단계대로 Firebase 프로젝트를 연결하면, 모든 기기에서 실시간으로 데이터가 공유돼요.

1단계 — console.firebase.google.com 접속 후 프로젝트 만들기
2단계 — 왼쪽 메뉴 빌드 → Realtime Database → 데이터베이스 만들기 (테스트 모드)
3단계 — 프로젝트 설정(⚙️) → 웹 앱 추가(</>) → SDK 설정의 값들을 아래에 입력
